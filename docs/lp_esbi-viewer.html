ESBI Viewer

ESBIクワドラント（従業員・自営業者・事業主・投資家）の収入構造を可視化するWebアプリケーション

Spring Boot 3.5.5 Java 17 Bootstrap 5 Thymeleaf

機能概要

ESBI Viewerが提供する主要機能

データ入力

各クワドラントの収入額を簡単に入力できます。直感的なフォームで効率的なデータ管理を実現。

可視化

収入構造をクワドラント形式で視覚的に表示。パーセンテージと金額で収入バランスを一目で把握。

履歴管理

過去の収入データを時系列で管理。収入構造の変化を追跡し、成長の軌跡を確認できます。

学習ガイド

ESBIクワドラントの概念や財務的自由への道筋を詳しく解説。理論を実践に活かせます。

ESBIクワドラントとは

4つの収入源を理解し、財務的自由への道筋を明確にします

Employee (従業員)

給与所得 - 時間を対価とした安定収入

Business owner (事業主)

事業からの不労所得 - システムが働く収入

Self-employed (自営業)

事業所得 - 専門性を活かした独立収入

Investor (投資家)

投資からの不労所得 - 資産が働く収入

機能詳細

各画面の機能とユーザーエクスペリエンス

ホーム画面

ESBIクワドラントの概要とアプリケーションへの導線を提供

  • 各クワドラントの説明とアイコン表示
  • 直感的なナビゲーション
  • レスポンシブデザイン対応
  • Bootstrap 5によるモダンなUI

ホーム画面のスクリーンショット

カード形式で各クワドラントを表示し、データ入力への導線を提供

データ入力画面

各クワドラントの収入データを簡単に入力

  • 4つのクワドラントの収入入力フォーム
  • 数値入力の検証とエラーハンドリング
  • 保存後の可視化画面への自動遷移
  • 各クワドラントの説明テキスト表示

データ入力画面のスクリーンショット

各クワドラントの収入額を入力する直感的なフォーム

可視化画面

ESBIクワドラントの収入構造を視覚的に表示

  • 4つのクワドラントをカラフルに表示
  • パーセンテージと金額の同時表示
  • 合計収入の表示
  • ホバーエフェクトによるインタラクティブUI

可視化画面のスクリーンショット

各クワドラントの収入構造を色分けして表示

履歴画面

  • 時系列での収入データ表示
  • 統計情報（総記録数、期間など）
  • 過去データの詳細確認
  • データ削除機能
